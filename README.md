## 🛡️ Troubleshooting

If you encounter issues while you run the software, review these common solutions.

### Battle.net Login Error
If the game refuses to connect, ensure your Windows 11 24H2 update is current. Outdated operating systems often create conflicts with the authentication service. Restart your computer and try again.

### Frame Rate Drops
Disable variable refresh rate features in your graphics card control panel if you notice stutter. The software works best when you set your monitor to native 4K resolution before you open the game.

### Installation Failures
Ensure you have at least 100 GB of free space on your drive. If the installer hangs, check your antivirus settings. Sometimes security software blocks the modification of game files. Add the installation folder to your exclusion list.
